# AE タイムショートリーダー

- Retas Studioの .sxf, .csv,　東映アニメーションの .xdts がサポートされており、csvタイムシートに変換するスクリプトです。
- CsvTimeSheetScript v1.5.jsxに合わせて、自動タイムリマップが可能。

*現時点の TimeSheetReader.jsx はCSVファイルへの変換のみ対応です。自動タイムリマップの部分は含まれていませんので、ご注意下さい*
